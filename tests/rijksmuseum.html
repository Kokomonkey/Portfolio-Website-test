An Abstraction in Code

The Grid System

Unlike the previous pen drawing, this image is constructed entirely of HTML <div> blocks placed on a 60x40 CSS Grid. It's a digital mosaic made of code.

Simplified Geometry

To render the complex Neo-Gothic architecture using only rectangular blocks, details had to be abstracted into simple geometric shapes, using colors to represent brick, stone, and slate.

CSS "Windows"

Instead of placing hundreds of tiny window divs, which would clutter the code, the window rows are simulated using CSS linear gradients applied as background patterns on the brick blocks.
